Setup Guide (English)

How to Setup Megaoke

  1. Prepare the Files: Extract "Sample files.zip" into your installation folder. This includes song files, banners, graphics, and sound.
  2. Open Settings: Click Settings or press F9.
  3. Select Database Path:
    • Click Browse and select the folder containing your song files (.idx, .dat, .megidx, etc.).
      • You can use the sample files or an empty folder to create your own database if you have many MIDI/Kar files.
  4. MIDI Controls: Go to the MIDI Controls tab and select your output device (VirtualMidiSynth, WinLive Synth, Sound Canvas, MIDI USB, etc.).
  5. Finalize: Adjust lyrics, metadata, or scoring settings and click OK.

Song Database Management

  • Add Songs: Press F10 to add MIDI (Single or Batch). Use the Language tab for categorization.
  • New Database: If using an empty folder, press F9. When prompted with "No Database Found," click Yes to create one.

Gabay sa Pag-setup (Tagalog)

Paano i-setup ang Megaoke?

  1. Ihanda ang mga Files: I-extract ang "Sample files.zip" sa installation folder (laman nito ang mga kanta, banners, at sounds).
  2. Buksan ang Settings: I-click ang Settings o pindutin ang F9.
  3. Piliin ang Database:
    • I-click ang Browse at piliin ang folder ng iyong song files (.idx, .dat, atbp).
      • Maaaring gamitin ang sample files o isang empty folder kung nais mong gumawa ng sariling database gamit ang MIDI/Kar files.
  4. MIDI Setup: Sa MIDI Controls tab, piliin ang iyong output device (VirtualMidiSynth, WinLive Synth, MIDI USB, atbp).
  5. Tapusin: Ayusin ang lyrics o scoring settings at i-click ang OK.

Pagdagdag ng Kanta

  • Mag-add ng Kanta: Pindutin ang F10 (Single o Batch). Gamitin ang Language tab para sa genre at wika.
  • Bagong Database: Kapag empty folder ang pinili, pindutin ang F9. I-click ang Yes kapag lumabas ang "No Database Found."
